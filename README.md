# linux-setup
Linux setup script for first initialize install OS

## Usage

```sh
git clone https://github.com/lucthienphong1120/linux-setup
cd linux-setup
bash setup.sh
```

## Support OS version

- [ ] Ubuntu 20.04
- [x] Ubuntu 22.04
- [ ] Ubuntu 24.04
